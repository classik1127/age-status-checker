# Age & Status Checker

A simple frontend project that validates user input and displays an age category
and status using JavaScript.

## Features
- Input validation
- Age categorization
- Student / Worker status selection
- Clean UI feedback

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript

## What I Learned

- Input validation

- Working with radio buttons
